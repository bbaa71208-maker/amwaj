# مركز أمواج الخليج — نظام المبيعات

## الملفات
- `index.html` — النظام الرئيسي
- `manifest.json` — إعدادات PWA
- `sw.js` — Service Worker (يعمل بدون إنترنت)
- `icon-192.png` / `icon-512.png` — أيقونات التطبيق

## تثبيت على الجوال
1. افتح الرابط في Chrome
2. اضغط **القائمة** ← **إضافة للشاشة الرئيسية**
3. يصبح تطبيقاً كاملاً ✅

## بيانات الدخول
- مدير: `admin / 1234`
- مبيعات: `sales / 1234`
